# MySQL to MongoDB ETL using Airflow

## Overview
This project demonstrates an end-to-end ETL pipeline that migrates data from MySQL to MongoDB using Python and Apache Airflow.  
The goal is to learn data engineering tools by building a real-world project.

## Tech Stack
- Python
- MySQL
- MongoDB
- Apache Airflow
- Git & GitHub

## Pipeline Flow
MySQL → Extract → Transform → Load → Validation → Monitoring

## Project Structure
src/
- extract/     # MySQL data extraction
- transform/   # Data transformation logic
- load/        # MongoDB load logic

## Status
- Work in progress – currently implementing MySQL extraction.
